The gradle-blender project is made of 3 Gradle tasks:

com.eowise.blender.fbxconv.FbxConv
Convert FBX files to G3DJ or G3DB.
com.eowise.blender.render.RenderImage
Render a Blender scene.
com.eowise.blender.render.RenderAnimation
Render a Blender animation.

FbxConv task

This task wrap the libGDX fbx-conv command line executable. You specify the files you want to convert, where to write the converted files, and the format in witch you want to convert the files.

For instance, the task below take all the .fbx files in the resources folder and convert them to the G3DB format in the assets/3d folder:

{% highlight groovy %} task fbxConv(type: com.eowise.blender.fbxconv.FbxConv) { convert fileTree('resources', { include: '*.fbx'}) into file('assets/3d') g3db() } {% endhighlight %}

You can use the above methods to configure a com.eowise.blender.fbxconv.FbxConv task.

Methods

convert(FileCollection files) or convert(FileTree files)
required

Set all the .fbx files to be converted. This method takes either a Gradle file collection or a Gradle file tree as a parameter.

into(File folder)
required

Set the folder where the converted files will be created.

g3dj() or g3db()
required

Use one of the above method to set the type of the output files, either G3DJ or G3DB.

flip() , pack() and other optionals methods

It is possible to set other fbx-conv flags with theses methods:

Method Description
flip()

Flip the V texture coordinates.

fbx-conv command line flag-f
pack()

Pack vertex colors to one float.

fbx-conv command line flag-p
maxVertices(String size)

The maximum amount of vertices or indices a mesh may contain.

fbx-conv command line flag-m <size>
maxBones(String size)

The maximum amount of bones a nodepart can contain.

fbx-conv command line flag-b <size>
maxBoneWeights(String size)

The maximum amount of bone weights per vertex.

fbx-conv command line flag-w <size>
