Resize images

Let's say you want to divide by 2 all PNG files in the images folder and all subfolders to a resized folder relative to each source image. In addition to that, you want to suffix all resized files with -small. Easy:

task resize(type: com.eowise.imagemagick.tasks.Magick) {
    convert 'images', { include '**/*.png' }
    into { relativeDir -> "images/${relativeDir}/resized" }
    actions {
        -background('none')
        inputFile()
        -resize('50%')
        outputFile {
            filename, extension -> "${filename}-small.${extension}"
        }
    }
}

As you can see below, each image will be resized in a resized folder relative to the source path. Moreover, each resized file is renamed.
