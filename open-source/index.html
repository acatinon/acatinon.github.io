gradle-blender

Easily automate Blender command line images and animations rendering with RenderImage and RenderAnimation tasks.

Use FbxConv task to automate FBX/Collada/Obj files conversion to more runtime friendly formats used by libgdx. This Gradle task internaly use fbx-conv command line.
